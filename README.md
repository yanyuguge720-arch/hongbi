# 红笔 🖊

小红书文案排版工具 —— 让你的笔记更好看。

## 功能

- 实时预览 Markdown 格式（**加粗**、*斜体*、~~删除线~~、#话题）
- 9 种文案模板（好物推荐、日常分享、教程攻略等）
- Emoji 快捷插入
- 三种排版风格（清爽、可爱、极简）
- 行距/对齐控制
- 字数统计与进度条
- 草稿管理（本地存储）
- 自动保存
- 撤销/重做
- 导出为高清图片
- 深色模式

## 使用

直接打开 `index.html` 即可使用，无需任何构建工具或服务器。

## 部署

推荐部署到 Vercel：

1. 将本仓库导入 Vercel
2. Framework 选择 "Other"
3. 部署完成即可获得公开链接

## 技术栈

纯 HTML + CSS + JavaScript，零依赖。
